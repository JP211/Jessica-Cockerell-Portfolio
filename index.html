Peak Destination Rentals: Streamline Your Next Ski Vacation

Peak Destination Rentals is an application designed to maximize your time on the slopes by improving the equipment rental process. Built with MERN by a team of five developers.

Create an account and rent skis, snowboards and other necessities. Set your difficulty level in your user profile and only see equipment relevent to your skill level. Currently designed to serve four locations in Colorado - perfect for the upcoming season!

Tindfur - Matchmaking for Pets!

Tindfur is a match making application for finding a new pet friend based on the concept of the popular dating app - Tinder. It can also be used by animal rescue organizations to rehome pets.

Tindfur is an application built by a team of four developers that uses node.js, express, handlebars.js and mySQL. To see a live demo of this project, check out the Github link below!

WikiVideo

Wikivideo was created by a team of four developers in order to add video to article searches. By using the Youtube and Wikipedia APIs, we were able to create a more visual experience while searching for articles on a specific topic.

The homepage includes the top trending video on Youtube and the top trending videos for each search are also displayed with the article. To see a live demo of this project, check out the Github link below!
